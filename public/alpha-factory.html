Alpha Factory
Both roles see the same live market stream.
The Trader spots patterns and writes hypotheses. The Coder quantifies them and deploys signals.
Teams compete live — regime shifts mid-game force everyone to adapt.
Market Streams→ Trader Spots Pattern→ Publishes Hypothesis→ Coder Implements Signal→ Deploys Live→ Regime Shifts→ Repeat × 3
🔬
Trader Role
Watch the live price stream. Spot patterns — momentum, mean reversion, volume spikes, range breakouts.

Write hypotheses in the Alpha Notebook. Be precise: state what signal, what threshold, what edge you expect. Publish to your Coder partner.

After a regime shift, you must hypothesize why your alpha decayed and propose a new one.
⚡
Coder Role
You receive the Trader's hypotheses in your inbox. Read the idea and implement strategy(ctx).

You have the full API: sma, ema, rsi, z-score, bollinger, momentum. Deploy your strategy and watch its live PnL.

When a regime shift hits, diagnose why PnL changed and redeploy a new strategy fast.

Strategy API Reference
ctx.prices — number[], latest last
ctx.returns — log returns[]
ctx.volume — volume[]
ctx.sma(w) — simple MA → number
ctx.ema(w) — exponential MA → number
ctx.rsi(w) — RSI 0–100 → number
ctx.zscore(w) — z-score → number
ctx.bollinger(w) — {upper,mid,lower}
ctx.momentum(w) — price Δ over w ticks
ctx.position — current pos (−100 to 100)
ctx.tick — current tick index
Return: {action: 'buy'|'sell'|'hold', qty: number}
Example (RSI fade):
function strategy(ctx) {
  var r = ctx.rsi(14);
  if (ctx.position === 0) {
    if (r > 70) return {action:'sell', qty:50};
    if (r < 30) return {action:'buy',  qty:50};
  }
  if (ctx.position > 0 && r > 55) return {action:'sell', qty:50};
  if (ctx.position < 0 && r < 45) return {action:'buy',  qty:50};
  return {action:'hold', qty:0};
}
