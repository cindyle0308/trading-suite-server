The Concept: Simple Moving Average Crossover

A moving average smooths out noisy price data. We use two: a fast SMA (10 bars) and a slow SMA (30 bars). When the fast line crosses above the slow line, momentum is turning upward — a BUY signal. When fast crosses below slow, momentum is turning down — a SELL signal. The logic: the recent average is now higher than the longer-term average, so the trend is accelerating upward.

How to Play

1

Watch the chart. The yellow line is SMA-10 (fast), the blue line is SMA-30 (slow).

2

When yellow crosses above blue → click BUY (LONG). When yellow crosses below blue → click SELL (SHORT).

3

Click CLOSE to exit your position when the signal reverses or you want to lock in profit.

4

You are scored on realised P&L over 2 minutes. Each position assumes 100 shares. Hints will guide you for the first few signals.

What to Watch For

Crossovers work best when the lines have been diverging — a strong crossover. Avoid entering when the two SMAs are nearly flat and weaving together (chop). In this simulator the market has clear regime shifts (trending vs flat) — learn to feel the difference.

// Called every tick — return your trade decision:
function strategy(ctx) {
  ctx.prices      // Float64Array — price history
  ctx.returns     // Float64Array — log return history
  ctx.sma5        // number — 5-period simple MA
  ctx.sma20       // number — 20-period simple MA
  ctx.sma50       // number — 50-period simple MA
  ctx.volatility  // number — 20-day realised vol (annualised)
  ctx.position    // number — current position (−100 to +100)
  ctx.tick        // number — tick index
  return { action: 'buy' | 'sell' | 'hold', qty: 10 }
}
